## Proyectos

### 1. Limpieza de Datos en SQL - https://ronaldfreire33-svg.github.io/Limpieza-De-Datos-SQL/ 
Fundamentos de tratamiento de datos reales: nulos, duplicados y estandarización de texto.

### 2. [Funciones de Ventana en SQL](https://ronaldfreire33-svg.github.io/Bloque_Ventanas_SQL/)
Uso de ROW_NUMBER, RANK y DENSE_RANK aplicados a un caso de nómina y reconocimiento salarial, con el razonamiento detrás de elegir cada función.

### 3. [Análisis de Mercado: Autos Usados en Ecuador](https://ronaldfreire33-svg.github.io/Analisis_De_Mercado_SQL/)
Análisis end-to-end con datos reales de PatioTuerca: limpieza, agregaciones, rankings y segmentación por kilometraje, con gráficos y conclusiones de negocio.
